Configuration import & export

CAN Frames configuration ›› Multiple buses configuration ››

While creating a multiple buses CAN configuration, using the 'CAN Configuration\Import' command , CANStream offers the possibility to use an existing single bus CAN configuration (*.xcc) for the current CAN bus.


Simply right click on the CAN bus node of the tree view that you want to set, select the 'CAN Configuration' item and then the 'Import' command . A file opening dialog will pop up in which you can choose the CAN configuration file to import.



On the other way around, the 'Export' command permits to export the CAN configuration of a particular bus in order to reuse it as a single CAN bus configuration.
