Control layout

CAN controller appearance options ››

The ‘View’ command of the tool bar contains functions to customize the appearance of the current CAN controller.



Whether the active panel is ‘Spy and Manual’ or ‘Cycle’ the corresponding commands group is enabled.  ‘Reset’ button restores all default layout options for both ‘Spy and Manual’ and ‘Cycle’ panels.
