Data Transmission

Spy & Manual mode ››

The ‘Start' button of the tool bar will be enabled only if an operational PCAN-USB device has been selected.


Click that button to start the data transmission. While the manual mode is operating, ‘Start' button is disabled. Click the ‘Stop' button to stop the manual mode operation and consequently the data transmission.


Those commands control both transmission and reception parts of the ‘Spy & Manual' mode.


Transmission panel of ‘Spy & Manual' control is divided in two parts: ‘Engineering' on the top and ‘Raw' data transmission at the bottom.
