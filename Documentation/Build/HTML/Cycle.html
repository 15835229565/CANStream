Cycle

Main interface ›› Menu strip ››

This menu contains all functions related to the cycle player.



New: Create a new cycle. Check the ‘Cycle creation’ section for more details.


Open: Load a cycle into the cycle player of the current CAN controller.


Set start/end points: Access to cycle starting and ending points control functions. Check the ‘Cycle playing’ section for more details.


Tools: Access to cycle creation tools. Check the ‘Cycle creation’ section for more details.
