Manual control grid columns

CAN controller appearance options ›› Grids contents control ›› Grid columns ››

Click ‘Spy and Manual’ to get the grid columns commands of this mode.



The menu is divided in two parts:


Messages Tx: for the transmission part of the manual control.


Messages Rx: for the reception part of the manual control.


The ‘Manual Tx’ contains columns control commands of the engineering data transmission grid.



Columns of the raw data transmission grid cannot be customized since all columns are needed.


The ‘Manual Rx’ menu is divided in two parts:



Engineering messages: containing columns control commands of the engineering data reception grid.



Raw messages: containing columns control commands of the raw data reception grid.
