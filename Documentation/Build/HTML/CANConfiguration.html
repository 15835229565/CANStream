CAN Configuration

Main interface ›› Menu strip ››

This menu contains all function related to the CAN configuration of the current controller.



New: Create a new CAN configuration.


Open: Open a CAN configuration file.


Edit: Edit the current CAN Configuration.


Import DBC: Import a DBC file and use it as current CAN configuration.


Raw messages Tx: Management of transmitted raw messages list.
