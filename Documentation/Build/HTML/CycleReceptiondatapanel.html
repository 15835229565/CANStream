Cycle Reception data panel

Cycle mode ›› Cycle playing ››

During cycle playing, data received are shown and decoded in the ‘Received data’ panel. As per the manual mode, this panel is split in two parts: raw data and engineering data.


In the panel ‘Raw data received’ received CAN frames and values of their bytes are shown. While the ‘Engineering data received’ shows decoded signal values of received frames.



As per the manual mode, columns and rows of those two grids can be customized using their contextual menus or the ‘View’ menu of the main menu strip.
