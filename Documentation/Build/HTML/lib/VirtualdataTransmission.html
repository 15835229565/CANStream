Virtual data Transmission

Spy & Manual mode ›› Data Transmission ››

Virtual CAN signal appear with a green color into the grid.



Since the value of a virtual CAN signal is calculated, its value cannot be set by the user.


Virtual CAN signal transmission can be turned on and off at any time, checking or unchecking the box .
