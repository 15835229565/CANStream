Cycle mode

Main interface ›› CAN Controller panel ›› CAN controller modes ››

In ‘Cycle’ mode CANStream is sending a pre-defined sequence of message. This mode allows simulation of real usage conditions in which values have a very high dynamic that couldn’t be replicated by a human.
