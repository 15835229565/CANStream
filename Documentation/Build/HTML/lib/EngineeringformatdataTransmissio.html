Engineering format data Transmission

Spy & Manual mode ›› Data Transmission ››

In order to send engineering data over the CAN bus a CAN configuration file has to be loaded. Once loaded, this CAN configuration is displayed in the ‘Engineering data transmission' grid.


Simply change the content of the ‘Value' cell to send a new value.
