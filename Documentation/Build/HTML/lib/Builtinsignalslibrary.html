Built-in signals library

Built-in signals ››

Click on a library item of the tree view to edit its properties or click the ‘Create new signals library’ button to create a new signal.



A built-in signals library has very few properties:


    • Library name: Name of library.


    • Comment: Description of the library.


    • Read only: Check that box if you want prevent unwanted changes in the library.


Make all changes you want and then click the ‘Modify’ button to apply your changes. In the case of the creation of a new library, ‘Modify’ button will become ‘Create’, just click it to create the library.
