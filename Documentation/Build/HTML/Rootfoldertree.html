Root folder tree

Data analysis ›› Data browser ››

The root folder tree shows the folder arborescence from the current root folder.



Click the ‘Open’ button to change the root folder. Root folder text box is actually a list that can be dropped in order to get the history of all root folder previously used and quickly switch among those root folders.



The last ten folders used as root folder are saved in this list.


Click a folder to explore its subfolders and all their child folders.
